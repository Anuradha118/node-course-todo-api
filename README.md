# node-course-todo-api
A REST api for a Todo application.

# Features
- API for user signup and signin.
- Users are authenticated using JWT.
- API to create todos.
- API to get all todos by the logged in user.
- API to update a todo
- API to delete a todo

# Developed with:
  - MongoDB
  - ExpressJS
  - NodeJS
